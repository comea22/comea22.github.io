# Summary

一邊學習網頁開發，一邊紀錄的小地方。

## Demo

[部落格｜程式散步 A calming walk in web dev](https://blog.yjliang.com)

## 功能

- 文章分類篩選
- 關鍵字搜尋
- 響應式設計
- 留言功能

## 技術棧

- [Astro](https://astro.build/) - 靜態網站生成器
- [Tailwind CSS 4](https://tailwindcss.com/) - 樣式框架
- [Lucide Icons](https://lucide.dev/) - icon 圖示
- [Giscus](https://giscus.app/) - 留言系統（基於 GitHub Discussions）
- [Cloudflare Pages](https://pages.cloudflare.com/) - 網站託管
